# ISO/IEC 23053:2022 - 머신러닝 프레임워크
> Update: March 9, 2026 / Designer: Brian Lee / 100K AX Expert Team

## 개요
ISO/IEC 23053은 인공지능 시스템에서 사용되는 머신러닝(ML) 접근법에 대한 프레임워크를 정의합니다.
ML 파이프라인의 각 단계, 역할, 데이터 흐름 및 품질 관리 방안을 체계적으로 제시합니다.

## ML 파이프라인 프레임워크

### 1단계: 문제 정의 (Problem Definition)
- 비즈니스 문제를 ML 문제로 변환
- 성공 지표(KPI) 정의
- 데이터 가용성 및 제약조건 파악
- **컨설팅 적용**: Use Case 정의 시 ML 적합성 평가

### 2단계: 데이터 수집 및 전처리 (Data Collection & Preparation)
- 데이터 소스 식별 및 수집 계획
- 데이터 정제, 변환, 통합
- 특성 공학 (Feature Engineering)
- 데이터 품질 검증
- **컨설팅 적용**: 데이터 인프라 성숙도 평가 기준

### 3단계: 모델 개발 (Model Development)
- 알고리즘 선택 및 설계
- 모델 학습 (Training)
- 하이퍼파라미터 최적화
- 교차 검증 (Cross-validation)
- **컨설팅 적용**: 기술 아키텍처 설계 참조

### 4단계: 모델 평가 (Model Evaluation)
- 성능 지표 측정 (정확도, F1, AUC 등)
- 편향성 및 공정성 평가
- 설명가능성 (Explainability) 분석
- 강건성 (Robustness) 테스트
- **컨설팅 적용**: 모델 검증 체크리스트

### 5단계: 모델 배포 (Model Deployment)
- 서빙 인프라 구축
- A/B 테스트 설계
- 카나리 배포/블루-그린 배포
- API 인터페이스 설계
- **컨설팅 적용**: MLOps 플랫폼 요구사항 정의

### 6단계: 모델 모니터링 (Model Monitoring)
- 성능 드리프트 감지
- 데이터 드리프트 감지
- 자동 재학습 트리거
- 이상 탐지 및 알림
- **컨설팅 적용**: 운영 모니터링 체계 설계

## ML 접근법 분류

### 지도학습 (Supervised Learning)
- 분류 (Classification): 이미지 인식, 스팸 탐지, 고객 이탈 예측
- 회귀 (Regression): 수요 예측, 가격 예측, 품질 점수 예측
- **적용 산업**: 전 산업

### 비지도학습 (Unsupervised Learning)
- 군집화 (Clustering): 고객 세분화, 이상 탐지
- 차원 축소 (Dimensionality Reduction): 특성 추출, 시각화
- **적용 산업**: 금융, 유통, 제조

### 강화학습 (Reinforcement Learning)
- 의사결정 최적화: 공정 제어, 물류 경로
- 자율 시스템: 로봇, 자율주행
- **적용 산업**: 제조, 물류, 게임

### 딥러닝 (Deep Learning)
- CNN: 이미지/영상 분석
- RNN/LSTM: 시계열 분석, 자연어 처리
- Transformer: LLM, 번역, 요약
- GAN: 이미지 생성, 데이터 증강
- **적용 산업**: 전 산업 (특히 비정형 데이터)

## 데이터 품질 프레임워크

### 데이터 품질 차원
1. **정확성 (Accuracy)**: 데이터가 현실을 정확히 반영하는 정도
2. **완전성 (Completeness)**: 필요한 데이터 항목의 존재 여부
3. **일관성 (Consistency)**: 데이터 간 모순 없는 정도
4. **적시성 (Timeliness)**: 데이터의 최신성
5. **유효성 (Validity)**: 데이터가 정의된 규칙 준수 정도
6. **고유성 (Uniqueness)**: 중복 데이터 없는 정도

### 데이터 품질 관리 프로세스
- 품질 프로파일링: 데이터 분포, 이상치, 결측값 분석
- 품질 규칙 정의: 비즈니스 규칙 및 기술 규칙 설정
- 품질 모니터링: 지속적 품질 지표 추적
- 품질 개선: 문제 원인 분석 및 시정 조치

## 모델 거버넌스

### 모델 레지스트리
- 모든 ML 모델의 중앙 집중식 관리
- 버전 관리 및 이력 추적
- 메타데이터 관리 (성능, 데이터셋, 파라미터)

### 모델 승인 프로세스
- 기술 검증: 성능, 강건성, 공정성
- 비즈니스 검증: ROI, 리스크 평가
- 윤리 검증: 편향성, 프라이버시, 투명성
- 운영 검증: 배포 준비성, 모니터링 체계

### 모델 라이프사이클 관리
- 개발 → 스테이징 → 프로덕션 → 폐기
- 각 단계별 게이트 및 승인 기준
- 자동화된 CI/CD 파이프라인
