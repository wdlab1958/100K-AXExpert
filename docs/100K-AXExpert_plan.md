## Executive Summary

WDLAB는 26년간 인터넷 보안 전문기업으로 성장해 온 경험과 전문성을 바탕으로, AI Transformation(AX) 컨설팅 사업 진출이 전략적으로 매우 타당하다. 2026년 대한민국 정부의 AI 산업 육성 정책과 제조업 AX(MAX) 지원 정책 확대에 따라 시장 기회가 확대되고 있으며, 특히 보안-AI 융합 분야에서 차별화된 경쟁력을 확보할 수 있다.

본 보고서는 WDLAB의 AX 컨설팅 사업 진출을 위한 10가지 핵심 분석 항목을 심층적으로 다루며, 실질적인 실행 방안과 전략을 제시한다.

## 2. 조직 구조 결정: 별도 법인 설립 vs 전문 사업부 신설

### 2.1 두 가지 방안 비교 분석

| 평가 항목 | 별도 법인 설립 | 전문 사업부 신설 |
|-----------|---------------|-----------------|
| **초기 투자 비용** | 높음 (5억~10억원) | 낮음 (1억~3억원) |
| **설립 소요 기간** | 3~6개월 | 1~2개월 |
| **브랜드 독립성** | 높음 | 중간 |
| **의사결정 속도** | 빠름 (독립 경영) | 중간 (모회사 협의) |
| **리소스 공유** | 제한적 | 용이 |
| **리스크 분리** | 완전 분리 | 연결 |
| **세제 혜택** | 벤처/창업 혜택 가능 | 제한적 |
| **투자 유치** | 용이 | 어려움 |
| **인력 채용 유연성** | 높음 | 중간 |
| **기존 고객 활용** | 별도 계약 필요 | 즉시 활용 가능 |

### 2.2 단계별 권장 방안

#### Phase 1 (Year 1): 전문 사업부 신설 권장

**근거:**
1. **시장 검증 우선**: 초기 1년간 시장 반응 및 사업 모델 검증
2. **리소스 효율성**: WDLAB의 기존 인프라, 인력, 고객 네트워크 활용
3. **리스크 최소화**: 초기 투자 비용 최소화로 실패 시 손실 제한
4. **빠른 시장 진입**: 1~2개월 내 사업 시작 가능

**권장 조직 구조:**
```
WDLAB
├── 기존 보안 사업부
│   ├── 모의해킹팀
│   ├── 보안컨설팅팀
│   └── 보안관제팀
│
└── [신설] AI Transformation 사업부 (AX Division)
    ├── AX 컨설팅팀 (5~7명)
    ├── AI 솔루션팀 (3~5명)
    └── AI R&D팀 (3~5명)
```

#### Phase 2 (Year 2~3): 별도 법인 전환 검토

**전환 조건:**
- 연매출 30억원 이상 달성
- 전담 인력 30명 이상 확보
- 독자 고객 기반 50개사 이상 확보
- 외부 투자 유치 기회 발생

**전환 시 구조:**
```
WDLAB Holdings (지주회사)
├── WDLAB (보안 사업)
└── 100K-AX Expert Co., Ltd. (AX 사업) ← 별도 법인
    ├── 컨설팅 본부
    ├── 솔루션 본부
    └── R&D 센터
```

### 2.3 결론

**권장안: 전문 사업부 신설 → 단계적 법인 전환**

| 구분 | 1차 (1년차) | 2차 (2~3년차) | 3차 (4년차~) |
|------|------------|--------------|-------------|
| 형태 | 사업부 | 사업부/법인검토 | 별도 법인 |
| 인원 | 10~15명 | 20~30명 | 50명+ |
| 매출 목표 | 10억원 | 30억원 | 70억원+ |
| 투자 | 자체 | 자체+외부검토 | 외부 투자 |
