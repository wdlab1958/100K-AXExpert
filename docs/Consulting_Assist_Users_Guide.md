 1. 플랫폼 개요

 1.1 시스템 소개

AI Consulting Assistant Platform은 기업의 AI 도입을 체계적으로 지원하는 종합 컨설팅 플랫폼이다. 본 플랫폼은 AI 전략 수립부터 구축, 운영까지 전 과정을 5단계 프레임워크에 기반하여 지원하며, AI 거버넌스 체계 수립 및 MLOps 구현 표준을 제공한다.

본 플랫폼의 핵심 특징은 다음과 같다:
- 5단계 컨설팅 프레임워크: 전략 수립 → 설계 정의 → 구축 → 확산 → 운영의 체계적 접근
- AI 에이전트 협업: 전략 분석가, 설계 전문가, ROI 분석가, 리스크 평가가, 보고서 생성가 등 5개 전문 에이전트의 협업
- 국제 표준 기반: ISO/IEC 38500, ISO/IEC 24030, ISO/IEC 42001 등 국제 표준 프레임워크 통합
- Human-in-the-Loop (HITL): 인간 전문가의 검토와 승인을 통한 신뢰성 확보

 1.3 사용 환경

- 권장 브라우저: Chrome, Edge, Firefox 최신 버전

 2. 시스템 접속 및 화면 구성

 2.1 로그인 및 접속

1. 웹 브라우저에서 플랫폼 URL에 접속한다.
   - 로컬 환경: http://localhost:8000
   - 운영 환경: 관리자에게 문의

2. 사용자 인증 후 대시보드 화면으로 진입한다.
   - 현재 버전에서는 인증 없이 접속 가능 (개발 단계)

3. 대시보드에서 기존 프로젝트를 선택하거나 새 프로젝트를 생성한다.

 4. 새 프로젝트 생성 - 상세 가이드

 4.1 개요

새로운 AI 컨설팅 프로젝트를 생성하기 위해 의뢰 기업의 기본 정보와 현황을 입력하는 페이지이다. 본 단계에서 입력한 정보는 이후 모든 분석의 기초 데이터로 활용되므로, 정확하고 상세하게 입력하는 것이 중요하다.

 4.2 입력 항목 상세 가이드

 4.2.1 기업 기본 정보

본 섹션에서는 컨설팅 대상 기업의 기본 정보를 입력한다.

                                                    예시: "ABC제조 AI 전환 컨설팅 2025"

 4.3 프로젝트 생성 절차

프로젝트 생성을 위한 단계별 절차는 다음과 같다.

[단계 1] 기본 정보 입력
1. 사이드바에서 "새 프로젝트" 메뉴를 클릭한다.
2. "기업 기본 정보" 섹션의 모든 필수 항목을 입력한다.
   - 프로젝트명, 기업명, 산업 분류, 기업 규모, 사업 개요
3. 입력 완료 후 "다음" 버튼을 클릭한다.

[단계 2] IT 인프라 정보 입력
1. "IT 인프라 현황" 섹션의 모든 항목을 입력한다.
   - 체크박스 항목: 클라우드 보유 여부, 데이터 웨어하우스, 데이터 레이크, GPU 서버, API 연동 가능 여부
   - 숫자 입력 항목: 서버 수, 레거시 시스템 수
   - 선택 항목: 클라우드 제공자, 보안 수준
2. 입력 완료 후 "다음" 버튼을 클릭한다.

[단계 3] 데이터 자산 정보 입력
1. "데이터 자산 정보" 섹션의 모든 항목을 입력한다.
   - 숫자 입력: 데이터 총량 (TB), 히스토리 데이터 기간 (년)
   - 슬라이더: 정형 데이터 비율 (0~1), 데이터 품질 점수 (1~5)
   - 체크박스: 데이터 거버넌스 체계 보유 여부
   - 다중 선택: 데이터 소스 목록
2. 입력 완료 후 "다음" 버튼을 클릭한다.

[단계 4] 인적 자원 정보 입력
1. "인적 자원 정보" 섹션의 모든 항목을 입력한다.
   - 총 직원 수, IT 인력 수, 데이터 사이언티스트 수, ML 엔지니어 수, 데이터 엔지니어 수
   - AI 프로젝트 경험 수, 교육 예산 비율 (%)
2. 입력 완료 후 "다음" 버튼을 클릭한다.

[단계 5] 재무 자원 정보 입력
1. "재무 자원 정보" 섹션의 모든 항목을 입력한다.
   - 연 매출 (억원), IT 예산 비율 (%), AI 투자 가능 예산 (억원), ROI 기대 기간 (개월)
2. 입력 완료 후 "다음" 버튼을 클릭한다.

[단계 6] 조직 준비도 평가
1. "조직 준비도 정보" 섹션의 5개 항목을 슬라이더로 평가한다.
   - 경영진 지원도, 변화 관리 역량, 혁신 문화, 부서간 협업 수준, 리스크 허용도
2. 각 항목을 1~5점으로 평가한다.
3. 평가 완료 후 "프로젝트 생성" 버튼을 클릭한다.

[단계 7] 프로젝트 생성 완료
1. 시스템이 입력 정보를 검증한다.
2. 검증 완료 시 프로젝트가 생성되고 프로젝트 ID가 발급된다.
3. 대시보드로 이동하여 생성된 프로젝트를 확인한다.
4. "제1장: 컨설팅 방법론" 또는 "제2장: 5단계 프레임워크" 메뉴로 이동하여 컨설팅을 시작한다.

 4.4 입력 시 주의사항

1. 필수 항목은 반드시 입력해야 한다. 필수 항목 미입력 시 프로젝트 생성이 불가능하다.
2. 숫자 입력 항목은 정확한 값을 입력한다. 추정치를 입력하는 경우 메모에 명시한다.
3. 슬라이더 항목은 현재 수준을 객관적으로 평가한다. 과대평가나 과소평가는 분석 결과에 영향을 준다.
4. 입력 중간에 "임시 저장" 버튼을 클릭하여 데이터를 저장할 수 있다. 임시 저장된 데이터는 나중에 이어서 입력할 수 있다.
5. 프로젝트 생성 후에도 일부 정보는 수정 가능하지만, 프로젝트명과 기업명은 변경할 수 없다.

 5.2 AI 성숙도 진단

 5.2.1 개요

AI 성숙도 진단은 기업의 현재 AI 역량 수준을 4대 영역에서 평가하는 과정이다. 진단 결과는 AI 도입 전략 수립의 기초가 된다.

4대 진단 영역:
1. 전략 및 비전 (Strategy & Vision)
2. 조직 및 역량 (Organization & Capability)
3. 데이터 및 기술 (Data & Technology)
4. 프로세스 및 거버넌스 (Process & Governance)

성숙도 레벨:
- Level 1 (초기): AI 활동이 산발적이며 특정 개인의 역량에 의존
- Level 2 (반복 가능): 최소한의 프로젝트 관리 체계 수립
- Level 3 (정의됨): 표준 프로세스 및 방법론이 전사적으로 문서화
- Level 4 (관리됨): 정량적 측정 지표(KPI) 기반 프로세스 관리
- Level 5 (최적화됨): AI가 비즈니스 전략의 핵심 동력

3. 진단 결과가 화면에 표시된다.

1. 진단 결과를 확인한다.

 6. 제2장: 5단계 컨설팅 프레임워크 - 단계별 상세 가이드

 6.1 개요

5단계 컨설팅 프레임워크는 AI 도입의 전 과정을 체계적으로 관리하기 위한 단계별 접근 방법이다. 각 단계는 명확한 목표와 산출물을 가지며, 순차적으로 진행된다.

5단계 개요:
1. 전략 수립 (Strategy): AI 비전 및 전략 수립, 성숙도 진단, 기회 발굴, 로드맵 수립
2. 설계 정의 (Define & Design): 상세 요건 정의, 기술 아키텍처 설계, 거버넌스 체계 수립
3. 구축 (Build & Implement): PoC 수행, AI 플랫폼 구축, 솔루션 개발 및 통합
4. 확산 (Scale & Deploy): 파일럿 운영, 변화 관리, 전사 확산
5. 운영 (Operate & Optimize): 모니터링 설정, 지속적 개선, 거버넌스 관리

 6.2 1단계: 전략 수립 (Strategy)

 6.2.1 개요

1단계에서는 고객사의 AI 성숙도를 진단하고, AI 적용 기회를 발굴하며, 중장기 AI 로드맵을 수립한다.

- AI 성숙도 진단 (4대 영역 평가)

- 전략 및 로드맵 수립

3. 3~5개의 전략적 목표를 설정한다.

[단계 3] KPI 정의
1. "핵심 성과 지표 (KPI)" 섹션에서 "KPI 추가" 버튼을 클릭한다.
2. 각 KPI의 이름, 목표값, 현재값, 측정 주기를 입력한다.
3. 전략적 목표와 연계된 KPI를 정의한다.

[단계 4] 단계별 계획 수립
1. "단계별 계획" 섹션에서 "Phase 추가" 버튼을 클릭한다.
2. 각 Phase의 이름, 기간, 주요 과제, 예상 투자, 기대 성과를 입력한다.
3. 일반적으로 3~5개의 Phase로 구성한다.

[단계 5] AI 로드맵 생성
1. "로드맵 생성" 버튼을 클릭한다.
2. AI 에이전트가 입력된 정보를 기반으로 상세 로드맵을 생성한다.
3. 생성된 로드맵을 검토하고 필요시 수정한다.

[단계 6] 저장
1. "저장" 버튼을 클릭하여 로드맵을 프로젝트에 저장한다.
2. 저장된 로드맵은 이후 단계에서 참조된다.
